
Job: Any program that you run from a bash prompt. If the program starts other processes (for example, a pipeline in a bash script),
those processes are part of the same job as the main program.

Lexer (or Lexical Analyzer): Checks the syntax of the inputed job. Separates the symbols into tokens.

Tokens: Symbol with meaning. It can be a program name, a parameter, a pipe or alikes.

Parser: Processes the tokens created by the lexer into command tables that will be executed by the executor.

Command table: struct that contains the tokens generated by the parser.
