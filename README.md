## 🚀 Getting Started

### 🔧 Installation

1. Clone the Minishell repository:
```sh
git clone --recursive https://github.com/San-tito/Minishell
```

2. Change to the project directory:
```sh
cd Minishell
```

3. Compile the dependencies:
```sh
make
```

### 🤖 Running Minishell
```sh
./minishell
```
